# Apparel Sales Analysis (4th Quarter 2020)

Short summary
- **Goal:** Analyze quarterly apparel sales data and answer the problem statement provided in `1716984875_sales_analysis_problem_statement.pdf`.
- **Files:** `AusApparalSales4thQrt2020.csv` (dataset), `solution.ipynb` (analysis notebook), problem statement PDF.

How to use
- Open `solution.ipynb` in Jupyter or JupyterLab to reproduce the analysis.
- Requirements: Python 3.8+, pandas, numpy, matplotlib/seaborn. Install with `pip install -r requirements.txt` if you create one.
